# 更新日志 CHANGELOG

本文件记录《本源币：重写经济的地基》每一个版本的变化。

版本号遵循[语义化版本](./docs/版本号规范.md)：`主版本.次版本.修订号`（例如 v1.2.0）。

格式说明：
- **新增**：补充了新的章节、小节或论述
- **修改**：对已有内容的改写、深化或调整
- **修正**：错别字、表述、格式等小问题的修补
- **移除**：删去的内容

---

## [v1.0.0] — 2026-06

首个公开发布的**传播版**。

### 新增
- 第七章新增 **7.5「学什么，怎么学：当教育不再为了筛选」**。该节承接 7.4（教育的准入公平），进一步探讨教育的**方式**：批判以刷题、应试、标准答案为核心的填鸭式教育，指出其真正功能是"筛选"而非"教育"，而筛选的根源是稀缺；当本源币体系消除了生存层面的稀缺（见第六章），筛选便失去地基，教育得以回到因材施教、为兴趣而学、从"学历制"走向"项目制"，能力由真实做成的事来证明，而非由文凭预先排序。
- 原 7.5（财政赤字）顺延为 7.6，原 7.6（第三条路）顺延为 7.7。目录与导航已同步更新。

### 关于本版本
本仓库为**传播版**，与作者的主仓库正文内容一致，但不包含指向作者个人渠道的引流信息。如果你希望支持作者，请通过本书"写在最后"一节或本仓库 README 中列出的联系方式（邮箱 / Telegram / Discord）与作者取得联系。
