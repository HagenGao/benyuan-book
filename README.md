## 📖 这本书在讲什么

人类今天比历史上任何时候都更懂经济——更多的理论、更强的工具、更海量的数据。可那些最古老的问题：失业的恐惧、无止境的内卷、不敢生孩子、周期性的经济危机、越拉越大的贫富差距——却几乎纹丝不动，甚至愈演愈烈。

这本书认为，根源在一个几千年来从未被认真质疑过的假设：

> **钱，只能是正数。**

全书从这一个被所有人继承、却从未被追问的前提出发，一步一步推演：如果让货币可以为负，会发生什么？它试图证明，一套允许为负的货币，比我们现在这套"必须为正"的体系，更公平、更高效、更稳定——并在此基础上，重新安排劳动、医疗、教育、创新、人口、治理等几乎所有领域。

它借用马克思的诊断，却拒绝马克思的药方；它尊重哈耶克对分散知识与价格的洞见；它不靠暴力，不靠中央计划，只松开那一个最小的假设，让市场自己长出公平来。
